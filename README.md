Install
-------

```bash
$ git clone git@github.com:asgrim/godeploy.git
$ cd godeploy
$ git checkout [version]
$ composer install --no-dev
$ bower install
```

Replace `[version]` with the version you wish to use.

Upgrade
-------

```bash
$ git fetch origin
$ git checkout [version]
$ composer install --no-dev
$ bower install
```

Replace `[version]` with the version you wish to use.
